== Description ==

This plugin is a content recommendation widget that increase the quality of your website, increase time on site, decrease bounce rate and make smart your website performance.
You can customize font, thumbnails, number of recommendations, and position of widget.
And one of the main features is monetization of content you generate.


= Disclaimer =

 This plugin does require low technical knowledge.



= Please Note =

This project is currently in Beta.




== Installation ==
Manual installation:

1. Upload `congoro_widget` folder to the `/wp-content/plugins/` directory

2. Activate the plugin through the 'Plugins' menu in WordPress

3. Click on the Congoro (کانگورو) from the Setting menu
Customize your widget and save to apply on your wordpress website.
